🔐 Setup SPIN

Usuarios y Permisos - Sistema de Gestión de Operaciones

1
Copiar el SQL
Aquí está todo el SQL necesario. Solo tienes que copiar y pegar en Supabase.
2
Ir al SQL Editor de Supabase
Abre el SQL Editor en tu proyecto.
3
Pegar y Ejecutar
1. Pega el SQL (Ctrl+V)
2. Click en el botón azul "RUN"
3. Espera a que diga "Success"
4
Iniciar servidor
En tu terminal, ejecuta:
npm run dev
5
Acceder a la app
Abre en tu navegador:
http://localhost:3000/login
